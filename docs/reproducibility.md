# Reproducibility Guide

## Minimal Reproduction

From the repository root:

```matlab
addpath(genpath(fullfile(pwd, "src")));
run("scripts/run_table1_cascaded_tanks.m");
```

This writes:

```text
results/generated/table1_cascaded_tanks_results.csv
results/generated/table1_cascaded_tanks_results.mat
```

## Full Main-Figure Reproduction

```matlab
run("scripts/run_rossler_main_figures.m");
```

Generated figures are written to:

```text
figures/exported/
```

The sensitivity sweep can be run with:

```matlab
run("scripts/run_sensitivity_analysis.m");
```

This script is long-running because it evaluates multiple hyperparameter grids.

## Traceability

The original paper scripts were retained in `scripts/legacy/`. The refactored Cascaded Tanks runner follows the same method-specific hyperparameters and fusion calls, but removes duplicated plotting and writes numerical outputs to CSV.

## Expected Outputs

The file `results/expected/table1_expected.csv` contains the Cascaded Tanks RMSE values generated by `scripts/run_table1_cascaded_tanks.m` on MATLAB R2020a with MATLAB's `smooth` function available. These values preserve the method-specific smoothing windows from the original legacy scripts. If `smooth_signal.m` falls back to `movmean`, small numerical differences may occur. If the manuscript table is updated, replace that file with the final accepted values.
